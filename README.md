Tetris
====

A 2D tetris game written in C++

Keyboard Settings
====

``← = Move To Left``
``→ = Move To Right``
``↓ = Move To Down``
``↑ = Transform``
``Space = Drop Down``

``F1 = Restart``
``F2 = Start/Pause``
``F3 = Speed Down``
``F4 = Speed Up``

``F5 = Difficulty Down``
``F6 = Difficulty Up``
``F7 = Music On/Off``
``F8 = Gridline Show/Hide``

``F9 = Exit``
``F12 = About``
